# 🌐 API Projects (JavaScript)

This repository contains two mini projects built using HTML, CSS, and JavaScript. The main purpose is to practice Fetch API, API integration, and displaying JSON data in a card layout.

## 📦 Projects Included

### 🔹 Project 1: Posts API
- Fetch posts data from API
- Display posts in card format

### 🔹 Project 2: Users API
- Fetch users data from API
- Display user details in card format

## 🚀 Features
- Fetch API integration
- Loading message while fetching data
- Error handling
- Card-based UI design
- Simple and clean CSS styling

## 🌐 API Source
https://jsonplaceholder.typicode.com/

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## 🎯 Purpose
This project is created as part of an assignment to improve understanding of APIs and frontend development.
